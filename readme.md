# Fakebookserver_v2
## _Express - Webpack_
## Development:

- Run webpack to build bundle
    ```sh
    yarn install
    yarn webpack
    ```
- Switch new Terminal
     ```sh
    yarn dev
    ```
## Production:

- Install
    ```sh
    yarn install
    ```
- Run app
     ```sh
    yarn start
    ```

## Config .env:
- .env
    ```sh
    NODE_ENV = 'development'
    PORT = 8080
    MONGO_DB_URL = mongodb+srv://username:password@cluster0.2vvnb.mongodb.net/database
    ```
